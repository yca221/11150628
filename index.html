📖 如何設定資料庫？

  1. 前往 Firebase 控制台，登入您的 Google 帳號。
  2. 點擊「新增專案 (Add project)」，輸入專案名稱後完成建立 (完全免費)。
  3. 進入剛建立的專案，在左側選單點擊 建構 (Build) > Firestore Database。
  4. 點擊「建立資料庫 (Create database)」，位置可選擇台灣 (asia-east1) 或保留預設值，點擊「啟用」。
  5. 在 Firestore 頁面中點擊上方的 Rules (規則) 頁籤，將內容修改為以下程式碼並點擊「發布」：
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /artworks/{docId} {
          allow read, write: if true; // 供教學快速測試用
        }
      }
    }
  6. 回到左上角點擊「專案總覽 (Project Overview)」旁的齒輪，或直接點擊畫面中央的 </> (網頁圖示)，註冊一個網頁應用程式 (名稱可自訂，如"藝廊")。
  7. 註冊完成後會出現一串程式碼，請找到 firebaseConfig 大括號內的 6 行設定 (從 apiKey 到 appId)，將這幾行選取並複製。
  8. 回到本系統，點擊右上角的 ⚙️ 設定 按鈕，將複製的內容貼上並儲存，即可成功連線！

💡 實用提示： 設定完成後，您可以在「設定」面板底部產生 QR Code，用大螢幕投影給學生掃描，他們就能直接免設定進入上傳畫面！
